  - ```w = {m, n}``` is concatenating ```m``` and ```n``` into ```w```
  - ```{m, n} = w``` is splitting ```w``` into ```m``` and ```n```
  - ```{N{m}}``` is duplicating ```m``` ```N``` times. ```N``` must be a constant.
